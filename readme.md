## Getting Started

Clone the repository.

```sh
git clone https://github.com/vitormalencar/
```

`cd` into the directory.

```sh
cd hooks react-memoization-hooks
```

Install the project dependencies:

```sh
yarn

# or

npm install
```

Start the development server:

```sh
yarn start

# or

npm run start
```
